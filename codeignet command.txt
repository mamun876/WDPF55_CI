## create project

***composer create-project codeigniter4/appstarter ci_project2

*php spark serve (to start the project in browser)
*php spark make:controller About  (to create new componenet files in controlller)
*php spark make:model ProductModel  (to create new componenet files in Model)
*php spark make:migration products1_migration    (to create new componenet files in   migration of database) after changing data of  migration the command will be (php spark migrate)

2 to add the component folder name in the view folder in the view folder
